# Traditional Time Clock - Modern Countdown & Wellness Platform

## Project Overview

Traditional Time Clock is a modern web application that combines ancient Chinese time culture with contemporary design principles. It features a beautiful countdown timer and comprehensive wellness guidance based on traditional meridian flow theory, designed specifically for Western users.

## Key Features

### 1. Real-time Zodiac Hour Display
- Dynamic display of current zodiac period (子, 丑, 寅, 卯, 辰, 巳, 午, 未, 申, 酉, 戌, 亥)
- Beautiful circular clock interface with real-time updates
- Current period highlighting with pulse animation effects

### 2. Advanced Countdown Timer
- **Custom Event Creation**: Set personalized countdowns for any event
- **Multiple Themes**: Choose from 5 beautiful themes (Default, Dark, Minimal, Colorful, Elegant)
- **Local Storage**: Save and manage multiple countdowns
- **Responsive Design**: Works perfectly on all devices
- **Real-time Updates**: Precise countdown with days, hours, minutes, and seconds

### 3. Zodiac Hours Explained
- Detailed descriptions and characteristics of each zodiac period
- Corresponding traditional Chinese medicine meridian information
- Wellness recommendations and precautions
- Recommended and avoidable activities for each period

### 4. Meridian Flow Theory
- Complete traditional Chinese medicine meridian theory
- Twelve meridian time correspondence table
- Energy flow pattern explanations
- Clinical application guidance

### 5. Wellness Guide
- Daily wellness principles
- Time-based wellness tips
- Seasonal wellness recommendations
- Personalized wellness plans

### 6. Modern Western Design
- Clean, minimalist interface following Western design principles
- Responsive layout optimized for all devices
- Touch-friendly interactions
- Accessibility features

## Technical Features

- **Modern UI Design**: Built with Tailwind CSS for a clean, professional look
- **Responsive Layout**: Perfect adaptation to various device sizes
- **Real-time Updates**: JavaScript-powered real-time calculations and displays
- **Modular Code**: Clear code structure for easy maintenance and extension
- **Performance Optimized**: Efficient DOM operations and smooth animations
- **Local Storage**: Persistent data storage for countdowns
- **Theme System**: Multiple visual themes for personalization

## Usage Guide

### Getting Started
1. **Direct Access**: Open `index.html` in any modern browser
2. **Local Server**: Use a local server for optimal experience
3. **Online Deployment**: Deploy to any static file web server

### Navigation
- **Home**: Real-time zodiac clock and platform overview
- **Zodiac Hours**: Detailed period explanations and wellness advice
- **Meridian Flow**: Traditional Chinese medicine theory and applications
- **Countdown**: Create and manage custom countdown timers
- **Wellness Guide**: Practical wellness advice and guidance
- **About**: Project information and technical details

### Countdown Timer Features
1. **Create New Countdown**:
   - Enter event title
   - Select date and time
   - Choose theme
   - Add custom completion message

2. **Manage Countdowns**:
   - Save multiple countdowns
   - Load previous countdowns
   - Delete unwanted countdowns
   - Preview themes before applying

3. **Themes Available**:
   - **Default**: Classic gradient design
   - **Dark**: Modern dark theme
   - **Minimal**: Clean, simple design
   - **Colorful**: Vibrant, energetic theme
   - **Elegant**: Sophisticated, professional look
